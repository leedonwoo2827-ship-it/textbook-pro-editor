# Textbook Pro-Editor (PDF Edition)

**AI가 교재의 오탈자, 맞춤법, 비문을 자동으로 찾아주는 교정 도구입니다.**

PDF 파일을 넣으면 AI가 읽고, 틀린 부분을 표로 정리해줍니다.  
설치가 필요 없고, 브라우저만 있으면 됩니다.

---

## 설치 및 실행 방법 (Windows)

### 1단계: 프로그램 다운로드

1. 이 페이지 상단의 초록색 **`<> Code`** 버튼을 클릭합니다
2. **`Download ZIP`** 을 클릭합니다
3. 다운로드된 ZIP 파일을 **D드라이브** 또는 원하는 폴더에 저장합니다

### 2단계: 압축 풀기

1. 다운로드된 ZIP 파일을 **마우스 오른쪽 버튼**으로 클릭합니다
2. **"여기에 압축 풀기"** 또는 **"압축 풀기"** 를 선택합니다
3. 압축이 풀리면 폴더 안에 `index.html` 파일이 보입니다

### 3단계: 프로그램 실행

1. 압축 푼 폴더 안의 **`index.html`** 파일을 **더블클릭**합니다
2. 크롬(Chrome) 또는 엣지(Edge) 브라우저에서 자동으로 열립니다
3. 아래와 같은 화면이 나타나면 성공입니다

> **반드시 인터넷이 연결된 상태**에서 사용해 주세요.

## Gemini API Key 발급 방법 (무료)

이 프로그램은 Google의 Gemini AI를 사용합니다.  
사용하려면 **본인의 API Key**가 필요합니다. (무료로 발급 가능)

### 발급 순서

1. 아래 링크에 접속합니다 (Google 계정 로그인 필요):  
   **https://aistudio.google.com/apikey**

2. 페이지가 열리면 **"API 키 만들기"** (또는 **"Create API Key"**) 버튼을 클릭합니다

3. 프로젝트를 선택하는 창이 뜨면, 아무 프로젝트나 선택하고 확인합니다  
   (프로젝트가 없으면 자동으로 만들어집니다)

4. 화면에 **긴 영문+숫자 조합의 키**가 표시됩니다  
   예) `AIzaSyB1234567890abcdefghijklmnop`

5. 이 키를 **복사** (Ctrl+C) 합니다

> **참고:**
> - 무료 티어로 충분히 사용할 수 있습니다
> - API Key는 개인당 최대 10개까지 생성 가능합니다
> - Key는 본인만 사용하세요 (타인에게 공유하지 마세요)

## 사용 방법

### 1. API Key 입력

- 프로그램 좌측 **"환경 설정"** 영역에서 **"Gemini API Key"** 입력란에  
  위에서 복사한 키를 **붙여넣기** (Ctrl+V) 합니다
- 한 번 입력하면 **자동 저장**되어, 다음에 다시 입력할 필요가 없습니다

### 2. 모델 선택

- 기본값 **"Gemini 2.5 Flash"** 를 추천합니다 (속도와 품질의 균형)
- 서버가 혼잡하다는 오류가 나면 **다른 모델**로 바꿔보세요

### 3. PDF 파일 업로드

- **"PDF 업로드"** 영역을 클릭하거나, PDF 파일을 끌어다 놓습니다
- 여러 파일을 한 번에 올릴 수 있습니다
- **파일당 최대 20페이지**까지 분석됩니다
- **텍스트 선택이 가능한 PDF만** 지원합니다 (스캔한 이미지 PDF는 불가)

### 4. 교정 분석 시작

- 우측 상단의 **"교정 분석 시작"** 버튼을 클릭합니다
- AI가 파일을 순서대로 분석합니다 (파일당 약 30초~1분 소요)
- 분석이 완료되면 우측 패널에 교정 결과가 표시됩니다

### 5. 결과 확인 및 다운로드

- 교정 결과는 표 형식으로 표시됩니다:

| 위치 | 수정 전 | 수정 후 | 비고 |
|------|---------|---------|------|
| Page 1 | 원래 문장 | 수정된 문장 (변경 부분 노란색 표시) | 수정 사유 |

- **"마크다운 다운로드 (.md)"** 버튼으로 결과를 파일로 저장할 수 있습니다

## 자주 묻는 질문 (FAQ)

### Q. "서버 과부하" 또는 "503" 오류가 나요
**A.** Gemini 서버가 일시적으로 바쁜 상태입니다.  
- 자동으로 3회까지 재시도합니다
- 계속 실패하면 **모델을 변경**해 보세요 (예: Gemini 3.1 Flash-Lite Preview)
- 또는 1~2분 후 **재시도 버튼**을 눌러주세요

### Q. PDF를 올렸는데 "텍스트를 추출할 수 없다"고 나와요
**A.** 스캔한 이미지 PDF는 지원하지 않습니다.  
PDF 파일에서 텍스트를 드래그해서 선택할 수 있는 파일만 사용 가능합니다.

### Q. API Key를 매번 입력해야 하나요?
**A.** 아닙니다. 한 번 입력하면 브라우저에 자동 저장됩니다.  
같은 컴퓨터의 같은 브라우저에서는 다시 입력할 필요가 없습니다.

### Q. 20페이지가 넘는 PDF는요?
**A.** 앞쪽 20페이지만 분석됩니다.  
긴 파일은 여러 개로 나누어 업로드해 주세요.

### Q. 인터넷 없이 사용할 수 있나요?
**A.** 아닙니다. AI 분석과 화면 표시 모두 인터넷 연결이 필요합니다.
